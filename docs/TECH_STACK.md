# Encore Tech Stack

## Backend
- Python 3.10+
- FastAPI (asynchronous API)
- PostgreSQL
- Redis (Caching, Message Broker)
- Celery (Background Jobs)
- Elasticsearch (Search)
- Docker & docker-compose

## Frontend
- Mobile: React Native
- Web: Next.js
- Admin Dashboard: React + Tailwind

## Cloud / Storage
- AWS S3 (Media Storage)
- CDN for static content
- CI/CD: GitHub Actions

## Algorithms
- Personalized feed based on user interactions (Likes, Comments, Saves)
- Recommendation system for artists & concerts
